# AI项目记忆

## 目标
- 独立于人类版本，构建 AI 自治交易治理实验项目（仅模拟盘）。
- 人类始终具备最高权限和应急控制权。

## 红线
- 仅允许 `SIMULATION`。
- 最大回撤、单日回撤、单笔亏损阈值必须持续监控。
- 凭据只允许环境变量，不允许明文入库。

## 当前版本
- v0.1.3
- 已完成治理闭环关键能力接入：RBAC、审批队列、候选评分与硬约束、灰度发布、快照回滚、审计与报告面板可视化。

## 更新记录 v0.1.3 - 2026-04-08
- 后端治理能力扩展：
	- 新增 Human Root 权限动作：冻结自治、人工回滚。
	- 新增治理配置更新与审批决策 API。
	- 候选策略采用统一评分函数 `J`，并在硬约束失败时短路拒绝。
	- 发布链路支持 champion-challenger 灰度扩量与自动晋升。
	- 审计事件、小时/日报写入统一治理载荷，支持前端检索回放。
- 前端治理看板扩展：
	- 新增“自治总控与目标红线”“审批队列”“实验与回测中心”“灰度发布中心”“回滚与审计中心”“日报/小时报告”等模块。
	- 支持在看板内对待审批项执行通过/拒绝。
- 验证结果：
	- `python -m pytest -q` 通过（5/5）。
- 风险与结论：
	- 已完成清单中治理核心链路，但 CI 闸门、密钥泄漏扫描、编排引擎（Temporal/Prefect）与统计显著性回测仍待后续阶段实现。
