# AI项目记忆

## 目标
- 独立于人类版本，构建 AI 自治交易治理实验项目（仅模拟盘）。
- 人类始终具备最高权限和应急控制权。

## 红线
- 仅允许 `SIMULATION`。
- 最大回撤、单日回撤、单笔亏损阈值必须持续监控。
- 凭据只允许环境变量，不允许明文入库。
